Projects

Shapeplane

Shapeplane is a generative art application written in Java using the Processing library. It creates vector imagery from a color palette and shape pattern that can be used as a basis for digital art. I created it in 2017 for my digital art class.

Adrift

Adrift is a first person space exploration game created in Unity and programmed in C#. I worked on it for a couple years, and while it never left the prototype stage I learned a lot and implimented a lot of reusable modular systems. Some of the major features implimented include player-programmable spaceships, an inventory system, and a quest system.

Dungeon Mom

Dungeon Mom is a game initially created for the 72 hour game jam Ludum Dare 46 with the theme "keep it alive". You play as an adventuring mother who cares for her newborn baby while fighting through waves of zombie enemies. It was created in Unity with C#. I worked on it with a friend who made the art and music.

Play in Browser

Sorting Race

Sorting Race is a program created for a group project in my algorithm engineering class. It demonstrates the speed differences between four sorting algorithms by running them side-by-side to sort strings of hex numbers. It was programmed to run in a browser using javascript and the p5.js library to display the race.
